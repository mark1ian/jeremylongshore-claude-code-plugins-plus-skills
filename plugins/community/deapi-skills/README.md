# deAPI Skills — AI Media Generation for Claude Code

AI media generation capabilities for Claude Code. Install once, then just describe what you want in natural language.

## Features

- **YouTube Transcription** — Transcribe any YouTube video by URL
- **Image Generation** — Text-to-image with multiple models (FLUX, Stable Diffusion, etc.)
- **Text-to-Speech** — Generate natural voice audio from text
- **OCR** — Extract text from images
- **Video Generation** — Text-to-video capabilities

## Installation

```bash
/plugin marketplace add deapi-ai/claude-code-skills
/plugin install deapi-skills@deapi-ai/claude-code-skills
```

Or clone directly:

```bash
git clone https://github.com/deapi-ai/claude-code-skills.git ~/.claude/skills/deapi-skills
```

## Setup

1. Get a free API key at [deapi.ai](https://deapi.ai) ($5 free credit, no card required)
2. Set `DEAPI_API_KEY` in your environment

## Usage

Just talk to Claude naturally:

- *"Transcribe this YouTube video: https://youtube.com/watch?v=..."*
- *"Generate an image of a cosmic landscape"*
- *"Read the text from this screenshot"*
- *"Convert this text to speech"*

## Compatibility

Works with Claude Code, Cursor, Windsurf, and Continue.dev.
